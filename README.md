
# Árbol Binario en C

Este es un ejemplo de un árbol binario implementado en el lenguaje de programación `C`. 




## Uso

Antes de compilar el código, necesitas asegurarte de tener un entorno de desarrollo C instalado. Recomendamos usar `Dev-C++`, un IDE de C/C++ que incluye el compilador GCC. 

> [!NOTE]\
> Puedes descargar `Dev-C++` desde el siguiente enlace: https://sourceforge.net/projects/orwelldevcpp/.

Sigue las instrucciones de instalación proporcionadas en el sitio web de Dev-C++.


## Copilación

Una vez que tengas `Dev-C++` instalado, puedes abrir el archivo "arbol_binario.c" en el IDE. Luego, sigue estos pasos para compilar el código:

- Abre el archivo "arbol_binario.c" en `Dev-C++`.
- Haz clic en **Compilar** en la barra de herramientas o utiliza la tecla F9.
## Ejecución

Después de compilar con éxito, puedes ejecutar el programa haciendo clic en **Ejecutar** en `Dev-C++` o simplemente ejecutando el archivo "arbol_binario.exe" generado.
## Menú de opciones

El programa ofrece un menú interactivo con las siguientes opciones:

- Ingresar nuevo nodo.
- Eliminar un nodo.
- Buscar un nodo.
- Mostrar recorrido en preorden.
- Mostrar recorrido en inorden.
- Mostrar recorrido en postorden.
- Contar nodos en el árbol.
- Mostrar el valor de la raíz.
- Mostrar los hijos izquierdo y derecho.
## Funcionamiento

El código implementa un **árbol binario de búsqueda**, donde los elementos menores se ubican en el subárbol izquierdo y los elementos mayores en el subárbol derecho. Puedes realizar diversas operaciones en el árbol, como insertar, eliminar y buscar elementos. Además, puedes mostrar el contenido del árbol en diferentes órdenes de recorrido.
